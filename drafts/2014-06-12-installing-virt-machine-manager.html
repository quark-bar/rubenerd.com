I first used VMM back on Fedora. I was trying out Xen for the first time, and wanted a GUI interface for messing around with it, ala VirtualBox. While I ultimately learned to use the shell instead, it was still fun to play with Xen in such an environment.

Trying out this afternoon on Debian, I almost unwittingly installed this instead:

# apt-cache search vmm
==> vmm - manage mail domains/accounts/aliases for Dovecot and Postfix

What we want is this:

# apt-get install virt-manager

I suppose it goes without saying you need Xen. Or maybe QEMU as an alternative?
