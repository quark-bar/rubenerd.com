How do I set up a failover IP?

OrionVM Support
posted this on December 3, 2012, 09:22

When you create an instance on OrionVM, you are usually assigned with two interfaces:

    A public internet connection, (in Linux eth0)
    A private network connection, (eth1), which only the VMs on your account can access (think of it as your own VLAN).

 
The network addressing on your private network connection is completely up to you- we assign the interface a (semi) random 10.0.0.0 /8 address, but you can change this at any time. In order to set up a failover IP for a private IP address, it's as simple as configuring your VMs for VRRP.
 
In order to do this for a public (internet) IP, you'll need to let us know beforehand so we can correctly configure our backend to allow multiple VM ownership of the same IP. It's worth mentioning that in this case you'll still need to configure VRRP to ensure only one of the two VMs uses the shared IP at a time.
