The Rubenerd Windows Vista review episode!

Ruben (Mac OS X and NetBSD user!) tries to objectively review Microsoft's latest desktop operating system: Windows Vista. He asks: Will it help you to be more productive? Has the Aero Glass interface been well executed? How much did Microsoft have on it's plate? Are the new features necessary? Are there any areas they've neglected? Can their market share save them? Microsoft has a lot of talented people, but is it really fully tapping their creativity? And a Windows XP security quote!
