There are plenty of more thorough lists online, but I thought I'd keep my own too. So far I've been using this software without problems:

  • Adobe Air 1.5.2
  • Simone’s Gimp 2.6.7
  • iMovie HD 6 from iLife 06
  • MacPorts 1.8.0 Snow Leopard release (guess that goes without saying!)
  • Mozilla Firefox 3.0.13
  • Mozilla Firefox 3.5.2
  • OpenOffice.org 3.1.1
  • TextMate 1.5.8
  • TweetDeck 0.26.6

Software I've been having problems with:

  • Skype 2.8.0.659 works as long as you do not thin the universal binary with ditto as I described in December 2007.

  • QuickTime 10 with Perian currently doesn’t support the MKV media container because of a registration incompatibility problem.

  • The Intel build of VLC 1.0 works, but for some reason takes upwards of 30 seconds on my machine to load and start playing MKV files. This wasn’t an issue on regular Leopard on the same machine.

  • XQuartz 2.4.0 returns an error saying "This update is designed for OSX Leopard Only.".

I've been able to compile the following ports from MacPorts without problems:

  • lame 3.98.2_1
  • ncurses 5.7_0
  • ncursesw 5.7_0
